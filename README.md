# M2 — E-commerce Frontend

## Descripción
Proyecto desarrollado como parte del **Portafolio del Módulo 3**, que implementa un **E-Commerce básico** utilizando **JavaScript** y **Bootstrap 5**.  
Incluye la lógica de **carrito de compras** (agregar/quitar productos, cálculo de total y descuento) y un flujo de **autenticación básica** mediante modal.

## Tecnologías
- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5 (CDN)

## Ejecutar el proyecto
- http://127.0.0.1:5500/ecommerce-frontend/
- https://abrifb.github.io/ecommerce/

## Funcionalidades
- Catálogo de productos usando objetos y arreglos
- Agregar productos al carrito
- Quitar productos del carrito
- Cálculo automático del total
- Aplicación de código de descuento **DESC15** (15%)
- Renderización dinámica del carrito en tabla Bootstrap
- Alertas visuales al agregar productos
- Inicio de sesión mediante Modal de Bootstrap
- Validación de contraseña con constante `PASSWORD_MAESTRA`
- Control de estado de sesión (`usuarioLogueado`)
- Cierre de sesión con alerta visual
- Actualización dinámica del Navbar según estado de autenticación
